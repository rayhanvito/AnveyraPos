# Anveyra POS

Anveyra POS is a SaaS POS platform for Indonesian retail UMKM: warung, toko kelontong, toko sembako, minimarket lokal, and simple retail shops.

Core direction:

- Flutter Android mobile POS, offline-first, phone and tablet friendly.
- Laravel API with PostgreSQL and Redis as the source of truth.
- Next.js customer dashboard for owners/admins.
- Next.js internal admin for tenant, billing, support, and audit operations.
- Indonesia-friendly flows: QRIS manual, transfer manual, shift cash reconciliation, IDR formatting, and clear Bahasa Indonesia copy.
